# BigDecimal
## ❓ Task
Given an array, s, of n real number strings, sort them in descending order.
Each number must be printed in the exact same format as it was read from stdin.
## ⬅️Sample Input
<pre>
9
-100
50
0
56.6
90
0.12
.12
02.34
000.000
</pre>

## ➡️Sample Output
<pre>
90
56.6
50
02.34
0.12
.12
0
000.000
-100
</pre>
